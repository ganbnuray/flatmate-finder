# LBA

Place all interview documentation and learnings here.

Suggested subfolders:
- `stage-1/` (before coding)
- `stage-2/` (after MVP 1)
- `stage-3/` (later)

Suggested file naming:
- `YYYY-MM-DD_<stage>_<participant-or-team-member>.md`
- or `YYYY-MM-DD_<stage>_summary.md` for synthesized findings.
