## 2. MVP 1 scope

**In:**
- Email/password auth (session-based)
- Profile: name, age, city, budget_min/max, housing_status, short bio, lifestyle fields
- Browse others (exclude self, exclude already-swiped)
- Like/Pass → mutual match → matches list
- Text-only messaging between matches

**Out (deferred):**
- Personality quiz, verification, filters, photos, maps, block/report, AI, payments, calls, mobile
- Housing status: store and display only in MVP 1; filtering/matching by status is a later decision
